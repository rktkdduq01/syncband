SyncBand API

SyncBand 백엔드 API 서버에 오신 것을 환영합니다. 이 서버는 SyncBand 애플리케이션을 위한 REST API 엔드포인트를 제공합니다.

주요 기능

  • 사용자 인증 및 프로필 관리
  • 실시간 음악 협업
  • 오디오 트랙 저장 및 관리
  • 커뮤니티 기능
  • 학습 컨텐츠 제공

API 문서

API에 대한 자세한 문서는 Swagger UI에서 확인하세요.

주요 엔드포인트

인증

POST /api/auth/register - 새 사용자 등록

POST /api/auth/login - 로그인 및 토큰 발급

사용자

GET /api/users/me - 현재 사용자 정보 조회

PUT /api/users/me - 사용자 정보 업데이트

트랙

GET /api/tracks - 모든 트랙 조회

POST /api/tracks - 새 트랙 업로드

WebSocket

WebSocket /ws/rooms - 실시간 협업 방 연결

WebSocket /ws/signaling - WebRTC 시그널링
